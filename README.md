This is simple discord bot which allows to student to ask get quiz questions.

# Usage
In discord server
```
/quiz
```

# Installation
Install dependecies:
- Python 3.11+
- Poetry

From source directory run
```
>export DISCORD_TOKEN='your token here'
>poetry run python bot.py
```
